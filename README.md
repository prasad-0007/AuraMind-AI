# 🧠 AuraMind AI - Future of Learning & Well-being
**Submission for Ignition HackVerse 2k26 | PVG’s COET&M, Pune**

AuraMind is an AI-powered adaptive learning assistant that uses **Real-time Computer Vision** to monitor a student's emotional well-being.

## 🚀 Key Innovation
Unlike static learning platforms, AuraMind detects **cognitive load and stress** through facial expressions. 
- If the user is stressed, the AI switches to **Marathi-localized support** and suggests a break.
- If the user is calm, it accelerates the curriculum.

## 🛠️ Technology
- **Engine:** Face-API.js (TensorFlow.js)
- **Frontend:** HTML5, Tailwind CSS, Space Grotesk Font
- **Deployment:** GitHub Pages (Serverless AI)

## 📍 Why AuraMind?
In the competitive environment of Pune's education hubs, student burnout is at an all-time high. AuraMind provides a **socially inclusive (Multilingual)** and **technically advanced** solution to mental wellness in education.
